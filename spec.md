# Gastrometric — Spec

## Goal
A food and nutrition tracking system that supports:
- recipe storage
- ingredient tracking
- meal planning
- nutritional analysis

## Core Entities
- Recipe
- Ingredient
- RecipeIngredient (join table)
- PantryItem

## Requirements
- Store recipes with structured ingredients
- Allow search by ingredient
- Track pantry inventory
- Support future nutrition calculations

## Non-Goals (for now)
- authentication
- UI polish
- external APIs
